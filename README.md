
# Floor Plan Project


## 1. 玄关

不要玄关

但需要以下功能：鞋柜，摆雨伞，摆钥匙，手套，临时摆买完的菜

入门右手边的墙部分可以打掉 

尝试计算 1米559 = 洗衣机宽 + 冰箱宽 + 柜子深度

能否摆得下洗衣机加冰箱 加定制柜子

洗衣机常见阔度 0.6米 深度 0.5米
单门冰箱阔度0.55米 深度 6.23米 东芝

这样子 玄关的柜子还可以有 0.4米的深度
